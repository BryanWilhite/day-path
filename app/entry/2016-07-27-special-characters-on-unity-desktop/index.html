Special Characters on Unity Desktop

On Windows I have become used to Alt-key combinations to get “special,” Latin characters. The Unity Desktop on Ubuntu has a unicode mode, invoked with Ctrl+Shift+u. When you are this mode, you have to enter a four-character code to render the desired glyph. These unicode codes are different from the corresponding Windows codes.
