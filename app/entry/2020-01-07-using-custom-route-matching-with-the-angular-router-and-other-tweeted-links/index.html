Overlooked pattern: using classes from a function component with useRef Hook.

function MyThing() {
  const ref = useRef()
  if (!ref.current) {
    ref.current = new MyClass()
  }
  // ...
}

class MyClass {
  // ...
}

Why? For imperative logic like jQuery plugins or logging.
